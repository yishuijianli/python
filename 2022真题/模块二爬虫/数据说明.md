## 📊 数据概览

### 书籍数据（来自豆瓣Top250）
- **数据量**: 25本书
- **数据来源**: 豆瓣读书Top250（真实爬取）
- **数据质量**: ✅ 真实可靠
- **更新时间**: 实时爬取

**包含字段**:
- 书籍名称：红楼梦、活着、哈利·波特、1984、三体全集等经典书籍
- 书籍评分：8.0-9.7分
- 书籍图片：豆瓣官方图片链接

### 评论数据（真实风格生成）
- **数据量**: 200条评论
- **数据来源**: 真实风格模拟数据
- **数据质量**: ✅ 适合做评论分析
- **评论分布**: 
  - 正面评论：70% (140条)
  - 中性评论：20% (40条)
  - 负面评论：10% (20条)

**包含字段**:
- 书名：15本热门书籍
- 作者：30个不同的用户名
- 标题：20种不同的评论标题
- 内容：真实风格的评论文本（正面/中性/负面）
- 评分：1-5星
- 日期：2024-2025年随机日期

## 💡 使用建议

### 推荐使用CSV格式
```python
import pandas as pd

# 读取数据
df = pd.read_csv('./data/book_comments.csv')

# 查看数据
print(df.head())
print(df.info())

# 评分分布
print(df['评分'].value_counts())

# 各书籍评论数
print(df['书名'].value_counts())
```

### 情感分析示例
```python
# 简单的情感分析
positive_words = ['好看', '推荐', '喜欢', '棒', '精彩']
negative_words = ['失望', '一般', '枯燥', '别扭']

def analyze_sentiment(text):
    if any(word in text for word in positive_words):
        return '正面'
    elif any(word in text for word in negative_words):
        return '负面'
    else:
        return '中性'

df['情感'] = df['内容'].apply(analyze_sentiment)
print(df['情感'].value_counts())
```

### 词云生成示例
```python
from wordcloud import WordCloud
import matplotlib.pyplot as plt

# 合并所有评论
text = ' '.join(df['内容'])

# 生成词云
wordcloud = WordCloud(font_path='simhei.ttf', 
                     width=800, height=400,
                     background_color='white').generate(text)

# 显示词云
plt.figure(figsize=(10, 5))
plt.imshow(wordcloud, interpolation='bilinear')
plt.axis('off')
plt.show()
```
